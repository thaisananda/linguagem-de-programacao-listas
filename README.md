# Linguagem de Programação

Projeto desenvolvido em **Java** com exercícios de lógica de programação.

## Conteúdo da disciplina

- Estrutura **Sequencial**
- Estrutura de **Decisão**
- Estrutura de **Repetição**
- **Modularização**
- **Recursividade**

## Objetivo

Praticar conceitos fundamentais de programação e desenvolvimento de algoritmos utilizando Java.
